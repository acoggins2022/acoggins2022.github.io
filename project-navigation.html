Project Case Study: Video Compressor GUI

A user-friendly desktop application for compressing video files with customizable settings.

Project Goal

This was a personal project I created to deepen my understanding of GUI development with Python and to learn how to integrate powerful command-line tools like FFmpeg into a user-friendly application. The goal was to build a standalone executable that could run on Windows without requiring any external installations.

Key Features

  • Easy to Use: A clean graphical interface built with Tkinter for selecting input files and output locations.
  • Customizable Compression: Users can adjust video quality (CRF), resolution, and audio bitrate to meet their specific needs.
  • Speed Control: The application exposes FFmpeg's encoding presets to allow users to balance compression speed and final file size.
  • Standalone Application: Packaged into a single `.exe` file that runs on Windows without needing Python or FFmpeg installed separately.

How to Use

The application is available for download on the project's GitHub page. Simply download the `VideoCompressor.exe` file from the latest release and run it. No installation is needed!
